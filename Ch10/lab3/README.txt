Answer to Question 10:
The CSS transition property controls how smoothly an element changes from one style to another. Instead of a hover effect happening instantly, a transition can make the change gradual. For example, if a navigation link uses transform: scale(1.1) on hover, transition: transform 0.3s ease; makes the link grow smoothly over 0.3 seconds.
