Bootstrap 4 Flex summary:
Bootstrap 4 Flex uses flexbox utility classes to control how elements line up and space themselves on a webpage. Classes such as d-flex, flex-row, flex-column, justify-content-center, justify-content-between, align-items-center, and flex-wrap can quickly arrange page content without writing much custom CSS. Flex is helpful for navbars, cards, buttons, and responsive rows because it makes it easier to align items horizontally or vertically and adjust layouts at different viewport sizes.
