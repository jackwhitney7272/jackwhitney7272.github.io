Bootstrap 4 Flex Summary

Bootstrap 4 Flex uses flexbox utility classes to quickly control webpage layout. Classes such as d-flex, flex-row, flex-column, justify-content-center, and align-items-center help align and space content without writing a lot of custom CSS. Flex utilities are useful for navigation bars, cards, buttons, and responsive rows because they can change how items line up at different screen sizes.
